# 电吉他自学教程 - 开发文档

## 项目概述

本项目是一个基于 HTML/CSS/JS 的电吉他自学教程，从最基础的知识开始讲解，逐步增加难度。使用 AlphaTab 开源库渲染交互式吉他六线谱，支持播放、调速、循环等功能。

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构，语义化标签 |
| CSS3 | 深色主题样式，响应式布局，CSS变量管理配色 |
| JavaScript (ES5) | 交互逻辑，兼容性优先 |
| AlphaTab v1.3.1 | 开源吉他谱渲染引擎，支持播放/循环/调速 |

## 架构设计

### 模块划分

1. **HTML层（index.html）**：教程内容、章节结构、谱例容器
2. **样式层（style.css）**：CSS变量主题系统、组件样式、响应式断点
3. **逻辑层（app.js）**：
   - 乐谱数据管理（Scores对象，AlphaTex格式）
   - AlphaTab渲染引擎封装
   - 播放控制（播放/暂停/停止/循环/调速）
   - 导航系统（侧边栏生成、章节折叠、滚动监听）
   - 状态持久化（localStorage）

### 设计模式

- **数据驱动渲染**：NavData 驱动侧边栏生成，Scores 驱动谱例渲染
- **状态集中管理**：AppState 全局状态对象
- **发布-订阅**：AlphaTab 事件监听（playerFinished等）

## AlphaTab 集成说明

### 版本信息
- 库名称：AlphaTab
- 版本：v1.3.1
- 项目地址：https://github.com/CoderLine/alphaTab
- 许可证：LGPL-3.0
- 文件大小：712KB（已压缩）

### 乐谱格式
使用 AlphaTex 文本格式定义乐谱：
- `\tempo` 设置速度（BPM）
- `\ts` 设置拍号
- `弦.品格` 表示音符（弦0=1弦高音E，弦5=6弦低音E）
- `|` 小节线
- `()` 同时发声（和弦）
- `s/h/p/b` 技巧标记

### 播放功能
- 使用 Web Audio API 合成音色
- 支持调速（30-200 BPM，步长10）
- 支持循环播放
- 播放时自动滚动跟随

## 兼容性

- Windows / Linux / macOS 全平台
- 支持 Docker 部署（静态文件，任意Web服务器）
- 所有路径使用相对路径
- 移动端响应式适配（768px断点）
