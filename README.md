# 🎸 电吉他自学教程

**从零基础到弹唱伴奏的完整电吉他学习平台**

一个基于 Web 的交互式电吉他自学教程，使用 AlphaTab 开源引擎渲染吉他六线谱，支持播放、调速、循环等丰富功能。包含 8 章系统化教程 + 150+ 首经典曲目练习。

## ✨ 功能特点

### 📚 系统化教程（8章内容）

| 章节 | 内容 | 难度 | 谱例数 |
|------|------|:----:|:-----:|
| **第一章** | 认识电吉他（构造、定音、姿势、调音） | ⭐ | 0 |
| **第二章** | 基础乐理与六线谱（音符、TAB读法） | ⭐ | 1 |
| **第三章** | 右手拨弦基础（拨片、交替拨弦） | ⭐ | 2 |
| **第四章** | 左手按弦基础（姿势、爬格子） | ⭐⭐ | 1 |
| **第五章** | 简单单音旋律（小星星、欢乐颂、生日歌） | ⭐⭐ | 3 |
| **第六章** | 强力和弦 Power Chord（两种指型、转换练习） | ⭐⭐⭐ | 1 |
| **第七章** | 节奏吉他入门（闷音Palm Mute、摇滚节奏型、Riff） | ⭐⭐⭐ | 2 |
| **第八章** | 进阶技巧入门（滑音、推弦、击弦、勾弦） | ⭐⭐⭐⭐ | 3 |

### 🎵 经典曲目练习（150+首 GTP 格式）

支持 Guitar Pro 格式（.gp3/.gp4/.gp5/.gpx），按难度分级：

- **🌱 入门级**：喜欢你、真的爱你（Beyond）
- **🎯 初级**：大声说喜欢你（灌篮高手）、海阔天空（Beyond）
- **⚡ 中级**：灰色轨迹、谁伴我闯荡（Beyond）
- **🔥 高级**：摇滚卡农（JerryC）、野蜂飞舞（古典改编）

- ▶️ **播放控制**：实时播放吉他谱音频

- ⏱️ **调速功能**：30-200 BPM 自由调节速度
- 📍 **进度跟踪**：蓝色竖线显示当前播放位置
- ✅ **练习清单**：自动保存练习进度到浏览器
- 📱 **响应式设计**：完美适配手机、平板、电脑

### 🎨 精美 UI 设计

## 🚀 快速开始

### 方式一：Node.js 本地服务器（推荐）

```bash
# 进入项目目录
cd 电吉他自学教程

# 启动服务器（端口 9999）
node server.js

# 打开浏览器访问
# http://localhost:9999
```

### 方式二：Docker 部署

```bash
# 构建 Docker 镜像
docker build -t guitar-tutorial .

# 运行容器（映射端口 9999）
docker run -d -p 9999:80 guitar-tutorial

# 访问 http://localhost:9999
```

> ⚠️ **重要提示**：由于 AlphaTab 使用 WebAudio 和 Web Workers API，**必须通过 HTTP 服务器访问**，不能直接用 `file://` 协议打开 HTML 文件。

## 📖 使用说明

### 浏览教程

1. **左侧导航栏**：点击章节标题展开/折叠内容
2. **滚动浏览**：滚动页面时自动高亮当前阅读位置
3. **进度条**：顶部显示整体学习进度

### 吉他谱操作

每个谱例都有完整的播放控制面板：

| 按钮 | 功能 | 快捷提示 |
|------|------|----------|
| ▶ 播放 | 开始播放音频 | 等待音色库加载完成 |
| ■ 停止 | 停止播放并复位 | 回到开头 |
| 🔁 循环 | 开启/关闭循环播放 | 练习难点时使用 |
| − / + | 调整播放速度 | 范围：30-200 BPM |

### 经典曲目页面

点击侧边栏「**经典曲目**」按钮或直接访问 `songs.html`：

- 按**难度等级**分组展示（入门/初级/中级/高级）
- 每首歌曲有独立的播放控制器
- 支持播放光标跟随（蓝色竖线 + 小节高亮）

### 练习检查清单

- 每个练习下方有勾选清单
- 勾选后**自动保存**到浏览器 localStorage
- 刷新页面后状态保持不丢失

### 移动端使用

- 点击左上角 **☰** 按钮打开导航菜单
- 点击菜单项后自动收起
- 所有控件针对触摸屏优化

## ⚙️ 自定义配置

### 修改主题颜色

编辑 `css/style.css` 中的 CSS 变量：

```css
:root {
    --accent: #e94560;      /* 主色调（红色） */
    --bg-primary: #1a1a2e;  /* 背景色（深蓝黑） */
    --text-primary: #e0e0e0; /* 文字色（浅灰） */
    /* 更多变量见 style.css 文件 */
}
```

### 添加新教程谱例

在 `js/app.js` 的 `Scores` 对象中添加：

```javascript
'my-exercise': [
    '\\instrument "Distortion Guitar"',
    '\\tempo 80',        // 速度 80 BPM
    '.',                // 乐谱开始标记
    '0.6 0.5 0.4 |'     // 弦.品格格式（注意顺序！）
].join('\n')
```

然后在 `index.html` 中添加对应的 HTML 容器。

### 添加新 GTP 曲目

在 `js/app.js` 的 `GtpFiles` 数组中添加：

```javascript
{
    file: 'gtp格式电吉他谱/我的歌曲.gp5',
    title: '我的歌曲',
    artist: '艺术家名',
    difficulty: '初级',      // 入门/初级/中级/高级
    description: '简要描述'
}
```

## 🔧 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| **吉他谱不显示** | AlphaTab 库文件缺失 | 确认 `js/alphaTab.min.js` 存在且完整（约 1.1MB） |
| **播放无声音** | 浏览器阻止自动播放 | 点击播放按钮手动触发；检查浏览器权限 |
| **样式错乱** | 缓存问题 | 清除浏览器缓存后强制刷新（Ctrl+F5） |
| **移动端菜单无效** | JS 加载失败 | 检查网络连接；确认 `app.js` 正常加载 |
| **GTP 文件加载失败** | 文件路径错误 | 确认文件存在于 `gtp格式电吉他谱/` 目录 |
| **中文文件名乱码** | URL 编码问题 | 已在 server.js 中处理 UTF-8 编码 |
