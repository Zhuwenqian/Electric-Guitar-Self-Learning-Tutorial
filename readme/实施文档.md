# 电吉他自学教程 - 实施文档

## 环境要求

- 任意现代浏览器（Chrome/Firefox/Edge/Safari）
- Node.js（用于启动本地服务器）
- **注意：由于 AlphaTab 使用 WebAudio 和 Web Workers，必须通过 HTTP 服务器访问，不能直接用 file:// 协议打开**

## 快速开始

### 方式一：Node.js 本地服务器（推荐）

```bash
# 进入项目目录
cd 电吉他自学教程

# 启动服务器
node server.js

# 访问 http://localhost:9999
```

### 方式二：Python 本地服务器

```bash
# Python 3
python -m http.server 8888

# 访问 http://localhost:9999
```

### 方式三：Docker 部署

```dockerfile
FROM nginx:alpine
COPY . /usr/share/nginx/html/
EXPOSE 80
```

```bash
docker build -t guitar-tutorial .
docker run -d -p 9999:80 guitar-tutorial
```

## 使用说明

### 浏览教程
- 左侧导航栏点击章节标题展开/折叠
- 点击小节标题跳转到对应内容
- 滚动页面时自动高亮当前章节

### 吉他谱操作
- **▶ 播放**：播放当前谱例的音频
- **■ 停止**：停止播放
- **🔁 循环**：开启/关闭循环播放
- **− / +**：调整播放速度（30-200 BPM）
- 已完成的章节会自动标记 ✓

### 练习检查清单
- 每个练习下方有检查清单
- 勾选后自动保存到浏览器本地存储
- 刷新页面后状态保持

### 移动端使用
- 点击左上角 ☰ 按钮打开/关闭导航菜单
- 点击导航链接后自动关闭菜单

## 自定义与扩展

### 添加新曲目

1. 在 `js/app.js` 的 `Scores` 对象中添加新条目：
```javascript
'my-song': [
    '\\tempo 80',
    '\\ts 4 4',
    '.',
    '0.0 0.2 0.3 0.5 | ...'
].join('\n')
```

2. 在 `index.html` 中添加对应的容器：
```html
<div class="tab-container" id="tab-my-song">
    <div class="tab-header">
        <span class="tab-title">曲目：我的歌曲</span>
        <div class="tab-controls">
            <span class="tempo-display" id="tempo-my-song">♩=80</span>
            <button class="btn btn-tempo" onclick="changeTempo('my-song', -10)">−</button>
            <button class="btn btn-tempo" onclick="changeTempo('my-song', 10)">+</button>
            <button class="btn btn-play" id="btnPlay-my-song" onclick="togglePlay('my-song')">▶ 播放</button>
            <button class="btn btn-stop" onclick="stopPlay('my-song')">■ 停止</button>
            <button class="btn btn-loop" id="btnLoop-my-song" onclick="toggleLoop('my-song')">🔁 循环</button>
        </div>
    </div>
    <div class="tab-render" id="render-my-song"></div>
</div>
```

### 修改主题颜色

编辑 `css/style.css` 中的 CSS 变量：
```css
:root {
    --accent: #e94560;      /* 主色调 */
    --bg-primary: #1a1a2e;  /* 背景色 */
    --text-primary: #e0e0e0; /* 文字色 */
    /* ...更多变量见文件开头注释 */
}
```

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 吉他谱不显示 | 确认 js/alphaTab.min.js 文件存在且完整（712KB） |
| 播放无声音 | 检查浏览器是否允许音频自动播放，尝试点击播放按钮 |
| 样式错乱 | 清除浏览器缓存后刷新 |
| 移动端菜单不工作 | 确认浏览器支持 ES5 JavaScript |

## 注意事项

1. 所有路径使用相对路径，确保跨平台兼容
2. AlphaTab 库已下载到本地，无需CDN
3. 用户进度保存在浏览器 localStorage 中，清除缓存会丢失
4. 音频使用 Web Audio API 合成，不同浏览器音色可能略有差异
