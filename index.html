第一章

认识电吉他

了解电吉他的构造、各部件的名称与功能，学习正确的持琴姿势和调音方法。

1.1 电吉他的构造

电吉他由琴头（Headstock）、琴颈（Neck）、琴身（Body）三大部分组成。与木吉他不同，电吉他依靠拾音器（Pickup）将琴弦振动转换为电信号，再通过音箱放大发声。

主要部件说明

部件名称位置功能说明
琴头 (Headstock)吉他顶端安装弦钮，固定琴弦一端
弦钮 (Tuning Pegs)琴头两侧调节琴弦张力，改变音高（调音用）
上弦枕 (Nut)琴头与琴颈交界固定琴弦间距，传递振动
琴颈 (Neck)琴头与琴身之间安装品丝，左手按弦区域
品丝 (Frets)琴颈上的金属条分割不同音高，按下琴弦时改变有效弦长
指板 (Fretboard)琴颈正面左手按弦的木质面板
品格 (Fret)品丝之间的区域从琴头方向数，第1格、第2格…
琴身 (Body)吉他主体安装拾音器、琴桥、电路等
拾音器 (Pickup)琴身弦下方将琴弦振动转为电信号（电磁感应原理）
琴桥 (Bridge)琴身尾部固定琴弦另一端，部分可调节弦高和音准
音量/音色旋钮琴身面板调节输出音量和音色明亮度
拨档开关琴身面板切换不同拾音器组合，改变音色
输出插孔琴身侧面或正面连接连接线到音箱
💡 小知识

电吉他的拾音器分为单线圈（Single Coil）和双线圈（Humbucker）两种。单线圈音色清脆明亮（如Fender Stratocaster），双线圈音色厚实温暖、噪音更小（如Gibson Les Paul）。

1.2 六根弦的标准定音

电吉他标准定弦（Standard Tuning）从最粗的弦（第6弦）到最细的弦（第1弦）依次为：

第6弦（最粗）: E2（低音Mi）
第5弦: A2（La）
第4弦: D3（Re）
第3弦: G3（Sol）
第2弦: B3（Si）
第1弦（最细）: E4（高音Mi）
🎵 记忆口诀

从粗到细：E-A-D-G-B-E（Every Adult Dog Growls Barks Eats / 每个成年狗叫吠吃）
从细到粗：E-B-G-D-A-E（Easter Bunny Gets Drunk At Easter / 复活节兔子在复活节喝醉了）

1.3 正确的持琴姿势

  • 坐姿：琴身凹槽放在右大腿上（右手弹奏者），琴颈向左上方倾斜约30-45度。
  • 左手：拇指放在琴颈背面中间位置，手指自然弯曲，用指尖按弦。
  • 右手：前臂自然放在琴身上，手腕放松，用拨片或手指拨弦。
  • 站姿：使用背带，琴身高度调整到坐姿时的高度，不要挂太低。
⚠️ 常见错误

左手拇指不要扣到指板上方（除非弹某些和弦需要），手腕不要过度弯曲，保持自然放松状态，避免腱鞘炎。

1.4 如何调音

每次练习前都要调音！可以使用电子调音器、手机APP调音器、或效果器内置调音器。

  1. 打开调音器，依次弹响每根空弦
  2. 观察调音器显示：指针偏左=音偏低（需拧紧弦钮），指针偏右=音偏高（需拧松弦钮）
  3. 目标：6弦=E, 5弦=A, 4弦=D, 3弦=G, 2弦=B, 1弦=E
  4. 新琴弦需要反复调几次，因为新弦会慢慢拉伸
第二章

基础乐理与六线谱

学习基本音符知识，掌握六线谱（TAB谱）的阅读方法，认识空弦音。

2.1 音符与半音

音乐中使用12个不同的音高，它们按半音关系排列：

C → C#/Db → D → D#/Eb → E → F → F#/Gb → G → G#/Ab → A → A#/Bb → B → C

  • 半音（Half Step）：相邻两个音之间的距离，在吉他上等于相邻1个品格
  • 全音（Whole Step）：两个半音，在吉他上等于相隔2个品格
  • #（升号）：升高半音
  • b（降号）：降低半音

注意：E和F之间、B和C之间是半音关系（没有E#或Fb），其余相邻音之间都是全音关系。

2.2 六线谱（TAB谱）的读法

六线谱是吉他专用谱，6条横线代表6根琴弦，线上的数字代表按下的品格数。

六线谱的基本规则

  • 最上面的线 = 第1弦（最细的弦，高音E）
  • 最下面的线 = 第6弦（最粗的弦，低音E）
  • 线上的数字 = 左手按在第几品格（0 = 空弦，不按）
  • 从左到右阅读 = 从先弹到后弹

第三章

右手拨弦基础

学习拨片（Pick）的正确使用方法，掌握交替拨弦技巧。

3.1 拨片（Pick）的正确使用

  • 握法：用拇指和食指侧面夹住拨片，拨片尖端露出约3-5mm
  • 角度：拨片与琴弦约成30度角，不要完全平行
  • 力度：轻轻握住，不要捏太紧，手腕放松
  • 厚度建议：初学者推荐0.7mm-1.0mm的中等厚度拨片

3.2 交替拨弦（Alternate Picking）

交替拨弦是电吉他最重要的右手技巧：下拨（Downstroke）和上拨（Upstroke）交替进行。

符号：𝄔 = 下拨（Down），𝄓 = 上拨（Up）

第四章

左手按弦基础

学习左手正确的按弦姿势，通过半音阶练习（爬格子）建立左手基本功。

4.1 左手按弦姿势

  • 拇指位置：放在琴颈背面中间，与中指相对
  • 手指编号：食指=1, 中指=2, 无名指=3, 小指=4
  • 按弦位置：手指尽量靠近品丝的右侧（靠近琴身方向），这样最省力
  • 指尖按弦：用手指尖（不是指腹）垂直按下，避免碰到相邻弦
  • 力度：刚好能让琴弦清晰发声即可，不要过度用力
⚠️ 注意

按弦太用力会导致：①音准偏高 ②手指疲劳 ③速度上不去。找到"最小有效力度"是关键！

第六章

强力和弦（Power Chord）

学习电吉他最核心的和弦类型——强力和弦（五和弦），这是摇滚、金属、朋克音乐的基础。

6.1 什么是强力和弦？

强力和弦（Power Chord），也叫五和弦，只包含根音（Root）和五度音（Fifth）两个音，没有三度音。

  • 为什么叫"强力"？因为没有三度音，不区分大小调，通过失真效果器后声音饱满有力、不浑浊。
  • 与木吉他和弦的区别：木吉他常用开放和弦（C、G、Am等，含3个以上音），电吉他失真后弹完整三和弦会非常浑浊，所以用强力和弦替代。
  • 可移动性：强力和弦的指型是固定的，只需在琴颈上平移就能得到不同音高的和弦，这是它最大的优势！
🎸 核心原理

强力和弦 = 根音 + 五度音。例如 C5 和弦 = C（根音）+ G（五度音）。标记为"X5"，如 C5、G5、A5、E5 等。

6.2 强力和弦的两种基本指型

强力和弦只有两种基本指型，分别以第6弦和第5弦为根音。学会这两种指型，你就能弹所有调的强力和弦！

第七章

节奏吉他入门

用强力和弦演奏节奏吉他，学习闷音（Palm Mute）技巧和基本的摇滚节奏型。

7.1 闷音（Palm Mute）技巧

闷音（Palm Mute，简称P.M.）是电吉他节奏演奏最核心的技巧！用右手掌侧（小鱼际）轻轻靠在琴桥附近的琴弦上，使声音变得短促有力。

  • 手的位置：右手掌侧轻触琴弦，位置在琴桥前方约1-2cm处
  • 力度控制：太轻=没效果，太重=完全闷死。找到"嚓嚓嚓"的短促有力感
  • 符号标记：谱上标注 P.M. 或虚线表示闷音段落
  • 应用场景：金属节奏、朋克节奏、摇滚riff的核心技巧
🎸 闷音的作用

闷音让强力和弦听起来更有节奏感和冲击力。对比一下：不闷音=嗡嗡嗡（持续），闷音=嚓嚓嚓（短促）。摇滚和金属的节奏吉他几乎离不开闷音！

第八章

进阶技巧入门

学习电吉他最基本的四种进阶技巧：滑音、推弦、击弦、勾弦。

8.1 滑音（Slide）

滑音是指弹响一个音后，手指不离开琴弦滑到另一个品格。符号：/（上滑）或 \（下滑）。

练习8-1：滑音练习（3品滑到5品）
♩=60

8.2 推弦（Bend）

推弦是电吉他最具表现力的技巧！按弦后向上推琴弦，使音高连续升高。符号：bend 或弯曲箭头。

  • 1/4推弦：推高半个音都不到，布鲁斯常用
  • 半音推弦（1/2）：推高1个品格（半音）
  • 全音推弦（Full）：推高2个品格（全音），最常用
⚠️ 推弦要点

推弦时用手腕旋转的力量，不是手指力量。拇指扣在琴颈上方作为支点。推弦后要能准确回到目标音高。

练习8-2：推弦练习（全音推弦 + 半音推弦）
♩=60

8.3 击弦（Hammer-on）

弹响一个音后，用另一根手指快速有力地敲击到更高品格的品丝上，使第二个音发声而不需要再次拨弦。符号：h。

练习8-3：击弦练习（3品击5品）
♩=60

8.4 勾弦（Pull-off）

与击弦相反：先同时按住两个品格，弹响后快速松开高品格的手指，让低品格的音发声。符号：p。

练习8-4：勾弦练习（5品勾3品）
♩=60
